# 🤖 Next Word Prediction using LSTM (PyTorch)

A Natural Language Processing (NLP) project built using PyTorch and Long Short-Term Memory (LSTM) networks to predict the next word in a sentence.

This project demonstrates the core concepts behind language modeling and text generation systems used in modern AI applications.

---

# 🚀 Features

✅ Text Tokenization

✅ Vocabulary Creation

✅ Word-to-Index Mapping

✅ Sequence Padding

✅ Word Embeddings

✅ LSTM Language Model

✅ Next Word Prediction

✅ NLP Pipeline Implementation

# 📂 Dataset

Training Sentences:

- i love ai
- ai is amazing
- deep learning is powerful
- python is awesome
- machine learning is fun
- i enjoy deep learning
- ai helps people
- deep learning uses neural networks

The model learns relationships between words and predicts the next likely word.
